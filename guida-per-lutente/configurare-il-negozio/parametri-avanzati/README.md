# Configurare i Parametri Avanzati

Esplorare il menu dei parametri avanzati è come guardare sotto il cofano di un’auto – qui trovi tutta la parte tecnica e i parametri per far funzionare correttamente il tuo negozio. Controllare le informazioni di configurazione, impostare i messaggi di posta elettronica, il backup del database: queste pagine non vengono visitate spesso \(esclusa la parte del database\), ma hanno una certa importanza, soprattutto per il tuo webmaster.
