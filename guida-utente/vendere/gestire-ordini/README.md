# Gestire gli Ordini

In qualità di gestore del negozio, dovrai affrontare un sacco di ordini, le relative fatture e le richieste di assistenza dei clienti - almeno questo è ciò che desideriamo per te. Il compito quotidiano di gestire numerosi ordini può essere scoraggiante. Fortunatamente PrestaShop fa del suo meglio per aiutarti a gestire con successo gli acquisti dei clienti, le note di credito e all'abbandono inevitabile del carrello.

Questo capitolo contiene le seguenti sezioni:

* [Ordini](ordini.md)
* [Fatture](fatture.md)
* [Note di credito](note-di-credito.md)
* [Bolle di Accompagnamento](bolle-di-accompagnamento.md)
* [Carrelli della Spesa](carrelli-della-spesa.md)
